# Plan: Add Support for QuiltNet-B-16 Patch Encoder

## Overview
Add support for QuiltNet-B-16 (ViT-B-16|GPT77), a CLIP-based histopathology model fine-tuned on the Quilt-1M dataset.

## Model Details
- **Name**: QuiltNet-B-16
- **Architecture**: ViT-B-16|GPT77 (CLIP model)
- **HuggingFace**: https://huggingface.co/wisdomik/QuiltNet-B-16
- **Embedding Dimension**: 512 (projection_dim from config)
- **Input Size**: 224x224
- **Preprocessing**: OpenAI CLIP normalization (mean=[0.48145466, 0.4578275, 0.40821073], std=[0.26862954, 0.26130258, 0.27577711])
- **License**: MIT (same as Quilt-1M)

## Implementation Approach

### 1. Create QuiltNetInferenceEncoder Class
**File**: `trident/patch_encoder_models/load.py`

Create a new encoder class `QuiltNetInferenceEncoder` that:
- Inherits from `BasePatchEncoder`
- Implements `_build()` method to load the model from HuggingFace
- Uses HuggingFace's `transformers` library to load `CLIPModel`
- Extracts the vision encoder and projection head
- Applies OpenAI CLIP normalization transforms

### 2. Implementation Details

#### Class Structure
```python
class QuiltNetInferenceEncoder(BasePatchEncoder):
    def __init__(self, **build_kwargs):
        super().__init__(**build_kwargs)

    def _build(self, with_proj=False, normalize=False):
        self.enc_name = 'quilt_b16'
        self.with_proj = with_proj
        self.normalize = normalize

        from transformers import CLIPModel, CLIPProcessor

        # Load model from HuggingFace
        model = CLIPModel.from_pretrained("wisdomik/QuiltNet-B-16")
        processor = CLIPProcessor.from_pretrained("wisdomik/QuiltNet-B-16")

        # Extract vision model
        vision_model = model.vision_model

        # Setup precision
        precision = torch.float32

        return model, eval_transform, precision

    def forward(self, x):
        # Extract embeddings using the vision encoder
        outputs = self.model.vision_model(pixel_values=x)
        pooler_output = outputs.pooler_output
        # Optionally apply projection head
        if self.with_proj:
            embeddings = self.model.visual_projection(pooler_output)
        else:
            embeddings = pooler_output
        # Optionally normalize
        if self.normalize:
            embeddings = embeddings / embeddings.norm(dim=-1, keepdim=True)
        return embeddings
```
